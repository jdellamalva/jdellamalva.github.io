# Project Title

My Portfolio

# Project Description

This is my portfolio, submitted as a project for the MIT xPRO Full Stack Web Development Course. It contains some links to my projects so far, and will be the place where I'll showcase all my future work to come.

# Roadmap for future improvements

I have so many plans to improve this portfolio namely;

- build a custom responsive dynamic modern sexy UI
- embed project demos as interactive iframes wherever possible
- create and link new projects here
- create a db connection so that I can pull project data dynamically rather than adding it to my code manually
- ...and so much more...

# Get Started

To get started, clone or fork this repo. In order to run the code you will need to ensure that you have nodejs installed, then run the terminal command http-server. Then check the portfolio out on your local host.
